====== Creating Lists in HTML ======
Created Wednesday 03 February 2021

You can create:
	- a numbered list	**<ol></ol>**
	- a bulleted list		**<ul></ul>**
	
	Within the **ul** or **ol** tags, you can create list items. 
	- You can also nest **ul **or **ol** tags inside of **ul** or **ol** tags
