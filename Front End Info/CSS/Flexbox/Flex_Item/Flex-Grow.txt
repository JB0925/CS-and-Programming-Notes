====== Flex-Grow ======
Created Wednesday 05 May 2021

Dictates how unused space should be spread between flex items.

Flex grow will make an item stretch to the space specified.

**Example:**
	flex-grow: 3;
	
	- this would make this item 3x as big as the other items.
