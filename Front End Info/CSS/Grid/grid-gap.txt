====== grid-gap ======
Created Monday 10 May 2021

Grid gap adds margin between each element in the grid.

{{~/Desktop/ScreenShots/Grid-Gap.png}}

The above has the following settings:
	- **grid-template-columns: 100px auto 100px;**
	- **grid-template-rows: 50px 50px;**
	- **grid-gap: 3px;**
