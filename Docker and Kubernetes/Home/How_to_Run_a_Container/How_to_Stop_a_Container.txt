====== How to Stop a Container ======
Created Tuesday 09 March 2021

run **docker stop CONTAINER_ID_NUMBER**

					**OR**

**docker stop CONTAINER_NAME (usually some silly name like "elastic_sanderson")**

type **docker ps** to ensure that the container isn't running
