====== How to Install Docker ======
Created Monday 08 March 2021

1). Go to www.docker.com/install

2). Find the instructions for your version in the navbar on the right: Linux

3). Follow the instructions from there.



**How to Verify that Docker was Installed:**
	- after installation, in the terminal, type **docker**.
	- brings up a long list of commands, type in **docker --version**
	
	
**NOTE: Docker **needs to be running from the **desktop** version as well if you are trying to run it in the terminal.
