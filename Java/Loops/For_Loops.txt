====== For Loops ======
Created Monday 14 February 2022

**Using an Index with a String:**

for (int i = 0; i < placeToStop; i++) {
	if (myString**.charAt(i) **== yourString**.charAt(i)**
	// do stuff...
}



**Using an Index with an Array:**

for (int i = 0; i < placeToStop; i++) {
	if (**arr[i] **== **arr[i+1]**)
	// do stuff...
}
