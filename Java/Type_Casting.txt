====== Type Casting ======
Created Monday 07 February 2022

To type cast in Java, we do so like this:

**(typeYouWant) (someValue / someOtherVal);**

**Example:**

(**byte) (myMinByteValue / 2);**

This casts an integer value to a **byte** data type.
