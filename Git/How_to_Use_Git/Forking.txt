====== Forking ======
Created Thursday 12 November 2020

Forking is like cloning, but you have access to make changes to the codebase when forking.

To Fork, go to "GitHub" and choose fork. 

When asked "Where should we fork PROJECT NAME?"
		**choose your user account**
