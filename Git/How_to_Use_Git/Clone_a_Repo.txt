====== Clone a Repo ======
Created Friday 27 November 2020

to clone a repo, run:

**git clone repo_url destination folder**


**Example:**
	git clone http://github.com/sgsgsw C:/Users/superuser
	

**Example2:**
	git clone http://github.com/somerepo .			#the "." means send it to the current directory
