====== Symbolic or Symlinks ======
Created Thursday 02 September 2021

**Syntax:**
	- **ln -s path/to/file.txt new_file.txt**
	

**Notes:**
	- moving or deleting the parent will cut the ties with the linked file
	- if you delete the original, the linked file will no longer exist
