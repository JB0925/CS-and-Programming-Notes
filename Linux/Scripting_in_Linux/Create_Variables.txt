====== Create Variables ======
Created Saturday 28 November 2020

create a variable on the command line with no spaces, like this: **foo=bar**

to print the variable, call **echo "$foo"	**returns **bar**

**NOTE:	**if you call **echo '$foo'** in single quotes instead of double quotes, it will print **$foo**
