====== cd - Change Directory ======
Created Thursday 12 November 2020

**cd / dirname					(no space)	#**takes you to the directory you specify, if it exists

**cd ..										#**takes you back one directory, or to the parent directory of the current directory

**cd [[../../../..file_or_folder]]						**#takes you back multiple directories into whatever file or folder you want to be in


**cd -										**# go back to the previous directory you were in (can work to toggle back and forth between two
													  # directories
