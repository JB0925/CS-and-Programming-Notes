====== mv ======
Created Thursday 12 November 2020

move or rename files with **mv**


takes two paths (old path and new path)


**Example:**
	**mv dotfiles.md foo.md			#**changes the name of the file
	
	**mv path/to/dotfiles.md new_path/to/dotfiles.md		**#changes the location of the file
