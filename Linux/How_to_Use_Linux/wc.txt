====== wc ======
Created Saturday 15 May 2021

Word Count

**wc <filename>**
	- returns **<# of lines> <# of words> <# of bytes>**


**wc -l <filename>**
	- returns only the number of lines in the file


**wc -w <filename>**
	- returns only the number of words in the file


**wc -c <filename>**
	- returns only the number of bytes
