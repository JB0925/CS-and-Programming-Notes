====== top and htop - Get Running Processes in Real Time ======
Created Tuesday 12 January 2021

the **top** command returns a list of all currently running processes in real time
also gives the amount of CPU usage each process is consuming, and how much memory it is consuming as well

press **q** or **Ctrl+C** to exit back to the command prompt
