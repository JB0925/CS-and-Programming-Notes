====== uname - gives the system information ======
Created Sunday 10 January 2021

{{.\pasted_image.png}}

uname gives the name of the system being run. In this case, Linux.

**uname -a** gives a lot more information, such as the name of the computer, when the system originated, name of the OS, etc.
