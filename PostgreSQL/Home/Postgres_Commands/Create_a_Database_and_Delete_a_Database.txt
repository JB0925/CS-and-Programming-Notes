====== Create a Database and Delete a Database ======
Created Tuesday 02 March 2021

running **CREATE DATABASE db_name_here;				**#make sure to end with a semicolon


**Delete:**
	- **DROP DATABASE db_name_here;**
	- be careful; data is gone forever
