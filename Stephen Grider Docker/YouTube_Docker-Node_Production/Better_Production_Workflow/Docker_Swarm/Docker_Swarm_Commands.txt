====== Docker Swarm Commands ======
Created Friday 28 January 2022

docker node ls
	- lists out all of the nodes present in the Docker Swarm


docker stack ls
	- lists out all of the current stacks
	
docker stack ps myapp
	- lists out all of the tasks that are running
