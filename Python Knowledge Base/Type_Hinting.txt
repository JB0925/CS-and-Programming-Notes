====== Type Hinting ======
Created Sunday 12 September 2021

**Constraining Types:**
	**Optional**
		- used to replace **None** references in your code
	
	**Union**
		- used to present a selection of types
	
	**Literal**
		- used to restrict developers to very specific values
	
	**Annotated**
		- used to provide additonal description of your types
	
	`**NewType**
		- Use to restrict type to a specific context
	
	**Final**
		- Use to prevent variables from being rebound to a new _{value}
