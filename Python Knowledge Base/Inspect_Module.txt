====== Inspect Module ======
Created Thursday 14 January 2021

{{.\pasted_image.png}}

The inspect module, and **inspect.getmembers / inspect.isclass**, is useful for getting information about classes and modules in general.

A class name would be like '**Formatter'**
Class info would be like **<class 'Formatter'>**
