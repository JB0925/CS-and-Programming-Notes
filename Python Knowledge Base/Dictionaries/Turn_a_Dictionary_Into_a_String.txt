====== Turn a Dictionary Into a String ======
Created Tuesday 02 February 2021

If you have a very nested dictionary (maybe JSON), an easy way to check if something is in the dictionary is to convert it to a string.

**Example:**
	- if 'my_string in str(dict):
		return True
	return False
