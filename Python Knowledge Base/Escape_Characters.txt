====== Escape Characters ======
Created Sunday 18 October 2020

**List of escape characters to use in python:**

**Code			Result**
	
\'				Single Quote	
\\				Backslash	
\n				New Line	
\r				Carriage Return	
\t				Tab	
\b				Backspace	
\f				Form Feed	
\ooo				Octal value	
\xhh				Hex value
