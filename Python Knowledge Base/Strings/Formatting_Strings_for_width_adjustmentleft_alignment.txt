====== Formatting Strings for width adjustmentleft alignment ======
Created Wednesday 04 November 2020

Use **str.format()**

**Example:**

**print('{0}). {1:<11} {2}'.format(i, name, country))**


i = counter variable
name = dict key
country = dict value
