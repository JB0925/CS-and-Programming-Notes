====== split() method ======
Created Sunday 26 July 2020

https://www.geeksforgeeks.org/python-string-split/


This will break a string into two or more strings, given a separator.

Example:

**For word in words:**
	 **word = word.split('.')**

So if word = 'code.html', after the code word = ['code', 'html']
