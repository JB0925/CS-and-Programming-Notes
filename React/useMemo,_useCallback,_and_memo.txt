====== memo  useMemo  useCallback ======
Created Tuesday 07 December 2021

React has three APIs for memoization / caching.

They are **memo, useMemo, useCallback**

React caching / memoization strategies only have a cache / memo size of 1. 

**They only keep around the most recent value.**
