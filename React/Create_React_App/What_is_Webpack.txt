====== What is Webpack ======
Created Sunday 20 June 2021

CRA comes with Webpack:

- **it enables us to import and export modules**
	- and it packages up all CSS/JS/images into a single file for the browser to use
	
- **it reloads the page whenever files are saved, so long as the server is running**
