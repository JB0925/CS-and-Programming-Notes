====== Set Image src in CRA ======
Created Thursday 24 June 2021

**Example:**
	**import myImage from "./myImage.jpg"**
	
	<div **className="Appname">**
		<img **src={myImage} className="Appname-myImage" alt="myImage"/>**
	</div>
