====== Components ======
Created Wednesday 02 June 2021

Components are the main building block of React.

They combine pieces of UI and JS logic.

Components can be classes that know how to render themselves into HTML.

**There are two types of components: function components, and class components.**

There can be many, many components, and even components inside of components.
