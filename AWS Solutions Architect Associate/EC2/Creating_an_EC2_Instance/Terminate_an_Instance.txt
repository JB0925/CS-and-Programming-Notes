====== Terminate an Instance ======
Created Sunday 09 January 2022

It is common when, you don't need an instance anymore, to back up data, and then terminate the instance.

In the **instances** dashboard, you can click **instance state**, then scroll to the bottom to **Terminate instance** in order to be able to terminate an instance.
