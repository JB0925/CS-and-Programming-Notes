====== General Purpose ======
Created Sunday 09 January 2022

The **General Purpose** Instance Type is:
	- Great for a diversity of workloads, such as web servers or code repos
	
	- It is a balance between:
		- Compute
		- Memory
		- Networking
	
	The **t2.micro** is a General Purpose EC2 instance.
