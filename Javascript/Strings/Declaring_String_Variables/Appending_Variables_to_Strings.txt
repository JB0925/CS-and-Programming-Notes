====== Appending Variables to Strings ======
Created Wednesday 17 March 2021

**Example:**
	var myString = 'FreeCodeCamp ';
	var anAdjective = 'is awesome!';
	
	myString += anAdjective;
	
	var someAdjective = "is worthwhile.";
	var myStr = "Learning to code ";
	
	myStr += someAdjective;
