====== Conditional Statements ======
Created Thursday 18 March 2021

**Conditional statements in Javascript involve IF, ELSE IF, ELSE**


**Example:**
	function threeChoices(num) {
	   if (num == 50) {
		return 'exact'
	}
	   else if (num > 10) {
		return 'large'
	}
	   else {
		return 'small'
	}
	}
