====== Case Sensitivity in Variables ======
Created Tuesday 16 March 2021

Variable and function names in Javascript, just like Python, **ARE case sensitive.**

Variable declaration is similar to class declaration in Python, in that variables are in camel case, i.e. **var myNameHere;**
