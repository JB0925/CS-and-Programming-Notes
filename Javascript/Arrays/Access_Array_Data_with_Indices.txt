====== Access Array Data with Indices ======
Created Thursday 18 March 2021

Just like Python, you can access an array value using bracket notation and an index value.

**Example:**
	**var ourData = [50, 60, 70];**
	**var myNumber = ourData[0];**
	
	**console.log(myNumber)**
		- returns 50
