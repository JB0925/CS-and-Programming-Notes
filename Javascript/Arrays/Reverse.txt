====== Reverse ======
Created Thursday 25 March 2021

**Reverse** modifies the array in-place, i.e. it does not make a copy; it modifies the original array.

**Example:**
	let myLst = [3, 5, 7, 9];
	myLst.reverse();
	console.log(myLst)
	
	**The above returns the same list, just sorted in reverse, i.e. [9, 7, 5, 3].**
