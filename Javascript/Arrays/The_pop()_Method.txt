====== The pop() Method ======
Created Thursday 18 March 2021

Works just like Python's **.pop()**

Removes the last item from the array and returns it.

**Example:**
	**var myArray = [1,2,3];**
		**newNum = myArray.pop();**
		**console.log(newNum);**
			returns **3**
		**console.log(myArray)**
			returns **[1,2]**
