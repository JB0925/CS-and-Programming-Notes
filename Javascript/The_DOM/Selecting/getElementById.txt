====== getElementById ======
Created Friday 16 April 2021

Selects an element by its ID and returns that object.


**Example:**
	const img = document.getElementById('bear-photo');
	
	If and only if there is a match, the object will be returned to you,
	
	If there is not a match, it will return **null**.
