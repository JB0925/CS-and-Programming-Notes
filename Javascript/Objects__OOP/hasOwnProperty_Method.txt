====== hasOwnProperty Method ======
Created Friday 19 March 2021

Every object has a **hasOwnProperty(property_name)** method.

It is a boolean construct.


**Usage:**
	**console.log(myObj.hasOwnProperty("property_name"));**
		returns **true** if it does have that attribute, and **false** if not.
