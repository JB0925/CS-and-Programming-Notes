====== Comparisons Using Logical Operators ======
Created Friday 19 March 2021

**And, or, not, etc.:**


**How to Use Logical And:**
	if (val <= 50 **&&** val >= 25) {
		do work
	  }


**How to Use Logical "or" Operator:**
	the "**or**" in Javascript is two of the **pipe** characters, or **| |**
	**if (val >= 50 | | val <= 100) {**
		**return true**
	}
