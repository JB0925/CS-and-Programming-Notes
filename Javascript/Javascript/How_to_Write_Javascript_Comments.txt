====== How to Write Javascript Comments ======
Created Tuesday 16 March 2021

**Inline comment:	/ / **your comment here
	**Example:**
		**var number = 5;  **/ / in-line comment


**Multi-line comment:**
	**Example:**
		**/ * This is a**
		**multi-line comment. * /**
