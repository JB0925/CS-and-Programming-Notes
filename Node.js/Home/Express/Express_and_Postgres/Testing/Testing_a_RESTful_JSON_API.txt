====== Testing a RESTful JSON API ======
Created Thursday 16 September 2021

**Example Routes:**
	GET		`/cats 	Displays all cats
	GET		`/cats/:id	Display a single cat
	POST	`/cats/:id	Create a cat
	PATCH	`/cats/:id	Update a cat
	DELETE	`/cats/:id	Delete a cat
