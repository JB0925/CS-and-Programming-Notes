====== Debugging Express ======
Created Friday 10 September 2021

Using the flag **--inspect-brk** will stop your code at the first line

To not stop at the first line in Express, use:
	**nodemon --inspect**
	
	Again, open **chrome://inspect** to pull it up in Chrome Debugger
