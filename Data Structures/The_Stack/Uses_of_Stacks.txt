====== Uses of Stacks ======
Created Wednesday 16 September 2020

Stacks are used everywhere, both in the actual writing of code as well as in real world scenarios

**A few examples:**
	- undo/redo button
	- back-paging on web browsers
	- recursion and recursive functions
