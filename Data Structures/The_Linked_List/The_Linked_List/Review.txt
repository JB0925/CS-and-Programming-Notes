====== Review ======
Created Sunday 20 September 2020

**Review:**
	- a Linked List is a sequential access linear data structure in which every element is a separate object called a Node, containing TWO parts:
		- the data
		- the **reference**, or **pointer**, to the next Node in the List
		
		
		{{.\pasted_image.png}}
		
	- **One Big Drawback of a Linked List:**
		- you can only ever go forward with the pointers, never backwards
