====== Create a Select Field in Your Form ======
Created Thursday 25 March 2021

{{.\pasted_image.png}}

To create a select field, just pass in a list or tuple of tuples of all the possible values (pay attention to first comment), and then choose **forms.Select** for the type of widget. In the args, pass in the list of tuples.
